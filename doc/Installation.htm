Installation

From source:
1. open a terminal in directory "moss"
2. sh make.sh
3. as superuser: sh install.sh

Dependencies

The following features outside of C89 are used:

Found in Feature
C99 Flexible array as a structs last member
C99 Fixed size types uint8_t, int32_t, uint32_t
C99 Supplements to math.h
C99 complex.h (import cmath, sf)
C11 CMPLX (complex.h)
Clang, GCC __builtin_add_overflow, __builtin_sub_overflow, __builtin_mul_overflow (if available)

Currently the Moss interpreter depends more or less on the following programs. 

# C99 compiler
apt-get install gcc

# alternative C99 compiler
apt-get install clang-3.5

# GNU Readline (optional)
apt-get install libreadline-dev

# GNU Multiple Precision Arithmetic Library
# (optional)
apt-get install libgmp-dev

# Graphics interface (optional)
apt-get install libsdl2-dev

# Vector font rendering for SDL,
# depends on FreeType
apt-get install libsdl-ttf2.0-dev

# Cross compiler, target: MS Windows
apt-get install mingw32

# GUI-library (optional)
sudo apt-get install libgtk-3-dev

Development tools 

# IDE
apt-get install geany

# Static program analysis
apt-get install splint

# Debugger
apt-get install gdb

# Dynamic program analysis
apt-get install valgrind
apt-get install kcachegrind

# Emulator for cross compiler output
apt-get install wine

# Compare a file and its modification
apt-get install kompare

apt-get install libgsl-dev
