About programming language Moss

Aim of the project: Development of a dynamic high level programming language, well suited for everyday use.

The working title of this project is "Moss".
Start of the project: October 2014.
Author: John Smith.
License: All files of the Moss project are released under the terms of the CC0 license.

This software is provided without any warranty. It is not recommended to use this implementation in safety critical applications of any kind.

The specification of the programming language and its standard library is a subject of research and thus can change from time to time. There is no standard, a stable infrastructure can be build upon.
