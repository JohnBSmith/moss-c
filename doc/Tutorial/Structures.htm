There are interesting, more complex data structures that are not supported in the core language. But the language provides the basic tools that are necessary for the construction of such data structures like linked lists or trees.

Singly linked list

What follows is a basic implementation of a singly linked list data type. 

sub List(a)
  y = table[List.type]
    first=null, last=null
  end
  for x in a
    y.push(x)
  end
  return y
end

List.type = table
  sub list
    t=self.first
    a=[]
    while not t is null
      a.push(t.value)
      t=t.next
    end
    return a
  end,
  
  sub str
    return "List(["+self.list().join(", ")+"])"
  end,

  sub push(x)
    if self.first is null
      self.last = table value=x, next=null end
      self.first=self.last
    else
      self.last.next = table value=x, next=null end
      self.last=self.last.next
    end
  end,

  sub shift(x)
    t = table value=x, next=self.first end
    self.first=t
  end,

  sub unshift
    if self.first is null
      return null
    else
      value=self.first.value
      self.first=self.first.next
      return value
    end
  end
end of table

It can be used as a queue data type. 

> q = List([])
> q.push(1)
> q.push(2)
> q.push(3)
> q.unshift()
1
> q.unshift()
2
