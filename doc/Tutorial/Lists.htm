Iterators and generators

An emitter is simply a function that has no arguments but returns a value each call until it is exhausted. There are infinite emitters that never exhaust. A function that returns an emitter is called generator. The most basic generator is iter, a function that obtains an iterator from an iterable object. An iterator is an emitter that returns the elements of a collection in their order. 

> i = iter([3,7,2])
> i()
3
> i()
7
> i()
2
> i()
# Exception: empty emitter

# An infinite iterator
> i = iter(1..)
> [i(),i(),i(),i()]
[1, 2, 3, 4]

# Take 4 elements
> i.list(4)
[5, 6, 7, 8]

# Iterator from a range
> i = iter(1..10)

# Iterator from a set
> i = iter(set([3,7,2]))
