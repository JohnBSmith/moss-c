Exception handling

In the following an exception of type Ea is thrown an then catched. 

Ea = object()

try
  raise table[Ea] value="Some message" end
catch e: Ea
  print(e.value)
end

The catch-part is only reached if the exception is of type Ea. There are two kinds of catch. 

# conditional catch:
catch e: exception_type

# unconditional catch:
catch e

The behavior of the conditional catch is described by the following algorithm. 

try
  # ...
catch e
  if e is in Ea
    print(e.value)
  else
    raise e
  end
end
